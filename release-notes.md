# Podvault v0.1.0

Initial public release for Obsidian Community Plugins.

## Features

- Parse Xiaoyuzhou podcast episode links
- AI transcription (OpenAI / Volcengine / DashScope) with local cache
- LLM-powered summaries, knowledge points, outlines, and quotes
- Built-in sidebar podcast player with timestamp navigation
- Optional Canvas mind map export

## Install

Available via Obsidian Community Plugins after review, or manual install from this release.

## Requirements

- Obsidian 1.4.0+
- BYOK: your own transcription and LLM API keys
